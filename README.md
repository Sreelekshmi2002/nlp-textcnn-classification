# TextCNN for Text Classification

This project implements a Text Convolutional Neural Network (TextCNN)
for sentence-level text classification.

## Overview
TextCNN is a deep learning model commonly used in NLP tasks such as
sentiment analysis and document classification.

## Key Features
- CNN-based architecture for NLP
- Word embedding layer
- Multiple convolution filters
- Max-pooling and softmax classification

## Technologies
- Python
- PyTorch / TensorFlow
- NLP preprocessing

## Use Case
Demonstrates deep learning fundamentals applied to natural language
processing problems.

## Status
Academic project – extended for learning and experimentation.
